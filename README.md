[![Travis CI build status](https://api.travis-ci.com/Jurasikt/dev_okvpn.svg?token=JAwKtvyvJKxrU4zYifzs&branch=develop)](https://travis-ci.org)

## Installation


```bash
git clone https://github.com/okvpn/okvpn-site-code.git
cd okvpn-site-code
composer install
```

And run migrations
```bash
cd applivation
phinx migrate
```
